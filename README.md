# Linux-Antivirus-Shell-Script

Using this application, a user can run a local scan which means that the virus database is locally available in the system and an advanced scan where the hash of the file is sent to virus total API so that it gives a detailed scan report of the file by more than 59 commercial antiviruses. Antivirus can be updated, which means that the local virus database is updated from an external link.

### Tech Stack Used : Shell Script
